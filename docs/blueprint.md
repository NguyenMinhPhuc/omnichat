# **App Name**: OmniChat

## Core Features:

- Embeddable Chatbot Interface: Provide a customizable chatbot interface that can be embedded on any website via a script tag.
- Knowledge Base Upload: Implement a knowledge base ingestion tool, which allows users to upload documents (e.g., PDFs, text files) containing the business's information.
- Intelligent AI Responses: Employ an AI to intelligently respond to user queries based on the content of the uploaded documents; it can use a tool to incorporate the data.
- Chatbot Appearance Customization: Enable customization of the chatbot's appearance (colors, fonts, logo) to match the website's branding.
- Display Chat History: Present conversation history.
- Next.js API Endpoint: Create a Next.js API endpoint to receive and respond to questions from the chatbot.

## Style Guidelines:

- Primary color: #29ABE2 (RGB). A bright, energetic blue, evoking trust and reliability, suitable for a chatbot interface.
- Background color: #F0F8FF (RGB). A very light blue to maintain a clean, professional look and provide good contrast.
- Accent color: #6495ED (RGB). A medium blue used for interactive elements, offering visual distinction without overwhelming the primary color.
- Font pairing: 'Poppins' (sans-serif) for headlines and short prompts, paired with 'PT Sans' (sans-serif) for longer chat body.
- Use simple, clear icons to represent different actions and options within the chatbot interface.
- Design a clean and intuitive chat interface with a clear distinction between user and chatbot messages.
- Use subtle animations to provide feedback to user actions (e.g., loading animations, message sent confirmations).
